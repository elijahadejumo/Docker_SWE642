# StudentSurveyApp

This project is an Angular based web application for collecting and managing student survey data for GMU's CS Department. The application allows the user to view and edit previous submissions.

## Hosting
The application runs locally on `http://localhost:4200/`.

## Development

The StudentSurveyApp was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.17.

## Features of the Application
- **Home**: Welcome Page with Navigation Cards
- **Survey Form**: Survey Form with validation including the required assignment 4 criteria
- **Survey List**: Displays all submitted surveys with an option to update or delete previous entries

## Technologies Used
- **Angular**: 17.0.0
- **TypeScript**: 5.2.2
- **BootStrap**: 4.6.2
